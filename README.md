# Food Ordering App

A simple Python-based food ordering application built as a Dataquest Guided Project. It demonstrates the use of dictionaries, custom functions, loops, and user input handling to simulate ordering food, managing a cart, and checking out.

## Features

- Display a menu of food items with SKUs and prices
- Add items to the cart with quantities
- Remove items from the cart
- Modify quantities of items already in the cart
- View the current cart with a subtotal, taxes, and total
- Checkout and end the ordering session

## How to Run

1. Open `food_ordering_app.ipynb` in Jupyter or VS Code.
2. Run all cells sequentially to initialize the app.
3. When prompted, enter a number (1–6) to choose an action and follow the on-screen instructions. Enter 6 to exit.

## Skills Practiced

- Working with Python dictionaries
- Defining and calling functions
- Using loops and control flow
- Handling user input
