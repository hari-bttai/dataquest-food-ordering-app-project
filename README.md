# Food Ordering App

This project is a simple Python-based food ordering application built as a Datquest Guided Project. It demonstrates the use of dictionaries, custom functions, loops, and user input handling to simulate ordering food, managing a cart, and checking out.

## Features

- Display a menu of food items with SKUs
- Add items to the cart with quantities
- Remove items from the cart
- Modify quantities of items already in the cart
- View the current cart contents
- Checkout and end the ordering session

## How to Run

1. Open the `Basics-3.ipynb` Jupyter Notebook.
2. Run the cells sequentially to initialize the app.
3. Follow the on-screen prompts to interact with the food ordering app.

## Skills Practiced

- Working with Python dictionaries
- Defining and calling functions
- Using loops and control flow
- Handling user input
